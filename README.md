# E-IDE —— 新一代智能体 IDE

> 用最简洁的架构，做最牛逼的产品。

---

## 产品简介

E-IDE 是一款**轻量化、AI 原生**的桌面集成开发环境，基于 TypeScript + Electron 构建，由青岛理工大学 2022 级水哥独立研发。项目借鉴了 VS Code、Trae 和 JetBrains 的设计理念，并在核心思路上大胆创新：**不做插件的堆砌，而是让 AI 成为 IDE 的一等公民**。

### AI 驱动的编辑体验

E-IDE 内置多模型 AI 助手（DeepSeek、GLM、Kimi、Qwen、GPT、Claude 等），支持在聊天中**直接修改项目文件**，修改完成后可通过**一键撤销**将代码回滚到修改前状态，同时将原始提问恢复至输入框——这套闭环在同类产品中独一无二。

CLI Skill 系统赋予 AI 读写项目文件、搜索代码、执行命令的能力，让 Agent 可以不依赖插件完成复杂的工程任务。用户右键即可将文件或文件夹添加到 AI 上下文，实现精准的局部代码分析。

### 现代化编辑器核心

编辑器采用 CodeMirror 6 引擎，支持 TypeScript、Python、Java、C/C++、Go、Rust、PHP、HTML、CSS、JSON、Markdown 等 **16 种语言的语法高亮**，提供自动补全、代码诊断、括号匹配、搜索替换等标准编辑能力。保持简洁白色背景，专注代码本身。

### 全栈环境管理

E-IDE 内置**软件一键安装系统**，可自动下载安装 Python、Node.js、Java、Anaconda 等开发环境。项目打开后自动检测本地运行时并生成运行文件列表，支持环境筛选。SSH 远程连接功能可在远程服务器上检测运行时并执行代码，实现本地开发与远程部署的无缝衔接。

### 完整功能矩阵

- **项目管理**：新建 / 打开 / 搜索本地项目，Git 推送
- **文件浏览器**：三栏布局，可拖拽调整面板大小
- **多 Tab 编辑**：切换、关闭、自动保存
- **运行系统**：自动检测运行时，匹配可执行文件
- **软件与插件市场**：内置下载器管理开发工具
- **设置面板**：语言切换，开发者信息展示

## 快速开始

```bash
npm install        # 安装依赖
npm run dev        # 开发模式
npm run start      # 构建并启动
```

## 技术架构

```
Electron 主进程 ─── 渲染进程（TypeScript + Vite）
    │                    │
    ├─ AI 服务            ├─ index.html（UI）
    ├─ LSP 服务           ├─ codemirror-setup.ts（编辑器）
    ├─ DAP 服务           └─ main.ts（应用逻辑）
    ├─ SSH 服务
    ├─ Git 服务
    ├─ 下载服务
    └─ 运行服务
```
